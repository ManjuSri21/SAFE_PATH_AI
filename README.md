# SafePath AI: Intelligent Urban Safety Navigator

SafePath AI is a next-generation urban safety navigation application designed to empower citizens with real-time safety insights, AI-driven routing, and community-driven hazard reporting.

## 🚀 Vision
To create a safer, more connected urban environment where everyone can navigate with confidence using real-time data and predictive AI.

## ✨ Key Features
- **AI-Powered Safe Routing**: Recalculates routes based on lighting, traffic density, and historical crime data.
- **Virtual AI Escort**: An active monitoring mode that provides high-precision tracking and AI voice guidance during your journey.
- **Environmental Safety Insights**: Real-time monitoring of weather, visibility (mist/fog), and traffic levels.
- **Community Hazard Reporting**: Instant reporting of poor lighting, suspicious activity, harassment, and more.
- **Dynamic Safety Heatmaps**: Visualize safety layers for crime risk and lighting levels directly on the map.
- **Panic/SOS System**: Instant alert system for emergencies with rapid contact notification.
- **Adaptive UI**: High-contrast light and dark modes designed for maximum legibility in all conditions.

## 🛠 Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Maps**: Leaflet.js
- **Icons**: Lucide React
- **Real-time**: Socket.io (Simulated for Demo)

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/safepath-ai.git
   ```
2. Navigate to the project folder:
   ```bash
   cd safepath-ai
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
